# Aldona Notes App

### Quick start

```bash
# Clone AldonaNotes
$ git clone 'repo'

# change directory to repo
$ cd AldonaNotes

# install the repo with npm
$ npm install

# install pods in 'ios' directory
$ cd ios
$ pod install
$ cd ..

# Run ios
$ react-native run-ios

# Run android
$ react-native run-android
```
